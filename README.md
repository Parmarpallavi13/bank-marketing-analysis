# 📊 Bank Marketing Data Analysis

## 📌 Project Overview

This project analyzes the bank marketing dataset to understand customer behavior and predict whether a client will subscribe to a term deposit.

---

## 🎯 Objective

To build a machine learning model that predicts customer subscription based on various features such as age, job, campaign details, and economic indicators.

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Seaborn
* Matplotlib
* Scikit-learn

---

## 📂 Dataset

* Dataset used: `bankmarketing.csv`
* Contains 41,000+ records and 20+ features

---

## 🔍 Steps Performed

1. Data Loading and Exploration
2. Checking Missing Values and Data Types
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Data Preprocessing (Encoding categorical variables)
6. Feature and Target Separation
7. Train-Test Split
8. Model Building using Random Forest
9. Model Evaluation using Accuracy, Confusion Matrix, and Classification Report

---

## 📈 Model Performance

* Accuracy: **91%**
* Model performs well for majority class (No)
* Lower recall for positive class (Yes) due to class imbalance

---

## 📊 Key Insights

* Dataset is imbalanced (more "No" than "Yes")
* Campaign and duration features are important for prediction
* Model can be improved using advanced techniques like SMOTE or hyperparameter tuning
