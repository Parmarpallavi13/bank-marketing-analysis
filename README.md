# 📊 Bank Marketing Analysis

## 📌 Project Overview
This project analyzes bank marketing data to predict whether a client will subscribe to a term deposit using Machine Learning.

## 🎯 Objective
To build a classification model that helps banks identify potential customers.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-learn

## 📂 Dataset
- Bank Marketing Dataset (CSV file included)

## ⚙️ Steps Performed
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training (Random Forest)
6. Evaluation

## 📈 Results
- Accuracy: **91%**
- Confusion Matrix and Classification Report generated

## 📊 Output
- Target distribution graph
- Model performance metrics
